🔴 Por qué falla
  • Usa 20 personas como universo — eso es lógica de Asamblea General
  • Aplica 55% × 20 = 11 mínimos — fórmula incorrecta para JD
  • Para JD el universo es 12 cargos y el mínimo es 7 fijo
  • El quórum se calcula por cargos válidamente ocupados, no por número bruto de asistentes
  • El dashboard muestra "· · ORGANIZACIÓN" — el tipo de órgano no se está leyendo
Reglas de Oro

Sin ambigüedad. Aplicar exactamente así.

"Para instalar una sesión de Junta Directiva se requieren mínimo 7 miembros presentes, entre principales o suplentes que actúen como principales, incluyendo el voto institucional de la Junta de Vigilancia."
✅ SÍ CUENTAN
  • Principal presente → +1 voto
  • Suplente actuando → +1 voto
    Solo si el principal NO está
  • Junta de Vigilancia → +1 voto total
    1 solo, sin importar cuántos miembros JV
❌ NO CUENTAN
  • Revisor Fiscal
  • Contabilidad
  • Invitados externos
  • Suplente si el principal está presente
  • JV como 3 votos — es 1 institucional
⚠️ Suplencias — Crítico
  • Cada cargo tiene su propio suplente específico. No existen suplencias cruzadas.
  • El suplente de Tesorería NO reemplaza Fiscalía.
  • El suplente del vocal Pérez NO reemplaza al vocal Penagos.
👉 "El voto pertenece al CARGO, no a la persona."
Regla Técnica

Copiar e implementar. El quórum se calcula por cargos válidamente ocupados, no por personas ni por asistentes brutos.

✅ Código correcto
// constantes — Junta Directiva ASOCOLCI
const TIPO_ORGANO = "JUNTA_DIRECTIVA"
const TOTAL_CARGOS_CON_VOTO = 12 // 11 JD + 1 JV institucional
const QUORUM_MINIMO = 7 // fijo — NO porcentaje

function calcularQuorum(cargos, JV_tiene_al_menos_un_presente) {
  let quorum = 0

  // contar cargos válidamente ocupados — no personas
  for (const cargo of cargos) {
    if (cargo.principal_presente === true) {
      quorum += 1 // principal suma
    } else if (
      cargo.principal_presente === false &&
      cargo.suplente_presente === true &&
      cargo.suplente_es_del_mismo_cargo === true
    ) {
      quorum += 1 // suplente válido del mismo cargo
    }
    // máximo 1 voto por cargo — nunca ambos
  }

  if (JV_tiene_al_menos_un_presente === true) {
    quorum += 1 // 1 voto institucional — NO 1 por miembro
  }

  return quorum
}

function validarQuorum(q, tipo_organo) {
  if (tipo_organo !== TIPO_ORGANO) throw new Error("Tipo de órgano incorrecto para esta regla")
  return q >= QUORUM_MINIMO // >= 7
}
